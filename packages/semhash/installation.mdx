## Base Installation
To install the SemHash package, you can use the following command:

```bash
pip install semhash
```

SemHash only depends on  [Model2Vec](https://github.com/MinishLab/model2vec) and [Vicinity](https://github.com/MinishLab/vicinity), making it extremely lightweight.
